# PhoneVerse P1 阶段汇报

## P1 已完成内容

- Contacts 可从桌面打开并返回桌面。
- 内置哥哥、妈妈、妹妹、同学、AI 助手 5 位联系人。
- 每位联系人拥有唯一默认会话，点击联系人可直接进入对应聊天。
- Chat 会话列表展示全部联系人及各自最新消息、时间和未读数。
- Dexie 数据库升级至 v2，新增持久化通知表，并兼容已有 P0 数据。
- 首次解锁后本地事件可延迟触发“妈妈”的预设消息；同一事件仅触发一次。
- 非当前会话收到消息时：
  - 保存新消息。
  - 增加对应会话未读数。
  - 创建持久化通知并进入 FIFO 横幅队列。
- 当前会话收到消息时仅追加消息，不显示横幅、不增加未读数。
- 通知横幅展示应用图标、联系人、摘要和相对时间，4 秒后自动收起。
- 点击通知可进入正确会话，并同步清除未读数、标记通知及消息为已读。
- 进入任意会话时会同步清除该会话的未读状态。
- 消息、通知、已读状态和未读数均由 IndexedDB 持久化，刷新后保留。
- P0 的锁屏、桌面、Chat 发送、本地固定回复与持久化功能保持可用。
- 未实现 Photos、Notes、Settings 或任何 AI/后端能力。
