# PhoneVerse P0 阶段汇报

## P0 已完成内容

- 初始化 React、Vite、TypeScript、Tailwind CSS、Framer Motion、Zustand、Dexie、Vitest、ESLint 和 Prettier 工程。
- 建立 Design Tokens、浅色视觉体系及 Apple-inspired 自有界面风格。
- 实现 `PhoneShell`：
  - 桌面端按 390 × 844 基准居中显示，并随可用视口等比缩放。
  - 移动端使用 `100vw × 100dvh` 全屏显示。
  - 统一管理外框、圆角、阴影、屏幕裁切和安全区域。
- 实现锁屏：
  - 当前时间和日期。
  - CSS 内置渐变壁纸。
  - 预设消息通知。
  - 上滑、点击及键盘解锁。
  - 使用本地 UI 状态恢复已解锁状态。
- 实现桌面：
  - 展示 Chat、Contacts、Photos、Notes、Settings 图标。
  - 当前仅 Chat 可用；P1、P2 应用保持禁用，不包含对应业务功能。
  - 支持打开 Chat 和返回桌面。
- 实现 P0 Chat：
  - 会话列表显示头像、昵称、最后消息、时间和预置未读数。
  - 可进入“哥哥”会话。
  - 支持发送非空文本、Enter 发送和消息自动滚动。
  - 消息通过 Dexie 持久化至 IndexedDB。
  - 本地规则在 800–1500 毫秒后生成固定回复。
  - 不包含 AI、通知事件引擎或 P1 未读状态同步。
- 建立 Web `DeviceService` 适配接口，未安装任何原生插件。
- 支持 `prefers-reduced-motion`，并为主要按钮、输入框补充可访问名称与键盘操作。
